Curso de Python para Ingeniería Petrolera

Visión del curso: Proveer una base sólida en Python, ideal para estudiantes y profesores que empiezan con la digitalización desde cálculos básicos hasta un nivel avanzado, aplicados en las diferentes áreas de la ingeniería petrolera: Perforación, Producción, Yacimientos con algoritmos para análisis de Datos y herramientas funcionales de forma interactiva para lograr un enfoque académico y profesional.

01
Introducción a Python
¿Qué es Python y por qué usarlo en ingeniería? Configuración e instalación. Primer programa en consola (print, entrada de datos).
Ejemplo práctico: scripts para convertir Unidades (Litros, Barriles, Metros, Pies, Pulgadas, Otros).
02
Operaciones Matemáticas y Manejo de Variables
Tipos de datos: enteros, flotantes, cadenas. Operadores matemáticos básicos.
Ejemplo práctico: Calcular porosidad, permeabilidad, Gravedad API a partir de valores ingresados.
03
Control de Flujo - Condicionales e Iteraciones
Estructuras condicionales (if, elif, else). Bucles básicos (for, while).
Ejemplo práctico: calcular la tasa de producción petrolera (Q) aplicando la Ley de Darcy.
04
Funciones
Qué son las funciones y cómo definirlas. Parámetros y retorno de valores.
Ejemplo práctico: Crear una función para calcular la presión hidrostática en un pozo de perforación.
05
Manejo de Archivos
Leer y escribir archivos de texto (.txt) y archivos en excel.
Ejemplo práctico: Leer archivos de producción y calcular la producción promedio de un conjunto de pozos desde un archivo y generar un resumen.
06
Listas y Diccionarios
Conceptos básicos: listas, diccionarios y sus aplicaciones.
Ejemplo práctico: almacenar datos de pozos (Nombre, Producción Diaria) y realizar cálculos simples.
07
Resolución de Problemas Básicos
Aplicar conceptos aprendidos para resolver problemas específicos.
Ejemplo práctico: Calcular la tasa de producción de un pozo petrolero usando la ecuación de declinación exponencial.
08
Proyecto Final + Simulación con las herramientas
Crear un script en Python para calcular el Índice de Productividad (IP), la tasa neta de producción de petróleo (Qo) y agua (Qw), además de determinar la Gravedad API de un crudo a partir de la gravedad específica (GEP).

Simulador de Parámetros de Perforación

Este simulador integra la ingeniería de petróleo con la ciencia de datos para crear un Gemelo Digital que modela el comportamiento dinámico de un pozo a 8,500 pies de profundidad. A través de potentes scripts en Python, la herramienta genera perfiles geológicos y operacionales precisos (incluyendo litología, presiones y desgaste de mecha) y emplea algoritmos de detección automática para identificar en tiempo real eventos críticos como Kicks, Washouts y Pegas Diferenciales. El sistema culmina en un Dashboard Interactivo en HTML que visualiza registros verticales (Well Logs) profesionales y KPIs de eficiencia, consolidándose como una solución educativa y técnica indispensable para optimizar el monitoreo y reducir los tiempos no productivos (NPT) en la industria actual.

Simulador de Producción

Es una herramienta de ingeniería de yacimientos que integra el Método de Vogel con algoritmos de Machine Learning para diagnosticar y predecir el comportamiento de pozos petroleros con empuje de gas en solución. El programa permite modelar la Curva de Oferta de Fluido (IPR) para determinar la tasa de producción óptima en función de la presión de fondo fluyente, pero va un paso más allá al emplear Regresión Lineal para aprender de los datos históricos y proyectar escenarios de producción futura. Además, la herramienta automatiza el análisis de declinación para identificar con precisión el Límite Económico del activo, presentando todos los resultados mediante gráficos comparativos interactivos que facilitan la toma de decisiones estratégicas para maximizar la recuperación de hidrocarburos y optimizar la vida útil del pozo.

Simulador de Yacimientos

Simulación interactiva que integra la ingeniería de yacimientos con entornos gráficos para modelar el comportamiento dinámico de fluidos en estructuras geológicas como anticlinales. A través de una interfaz que permite configurar parámetros críticos —como porosidad, permeabilidad, viscosidad y saturaciones de fluidos— para observar en tiempo real cómo interactúan las fuerzas del subsuelo durante procesos de producción. Al generar gráficos esta herramienta se convierte en un recurso educativo que transforma conceptos teóricos de flujo fraccional y caracterización petrofísica en una experiencia práctica y visual, ideal para optimizar el aprendizaje.
