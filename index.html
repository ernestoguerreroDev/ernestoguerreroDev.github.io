Curso de Python para Ingeniería Petrolera

Estructura completa basada en necesidades reales de la industria: desde fundamentos hasta simulación interactiva.

01 · Introducción a Python
¿Qué es Python y por qué usarlo en ingeniería? Configuración e instalación. Primer programa en consola (print, entrada de datos).
Ejemplo práctico: scripts para convertir Unidades (Litros, Barriles, Metros, Pies, Pulgadas, Otros).
02 · Operaciones Matemáticas y Manejo de Variables
Tipos de datos: enteros, flotantes, cadenas. Operadores matemáticos básicos.
Ejemplo práctico: Calcular porosidad, permeabilidad, Gravedad API a partir de valores ingresados.
03 · Control de Flujo - Condicionales e Iteraciones
Estructuras condicionales (if, elif, else). Bucles básicos (for, while).
Ejemplo práctico: calcular la tasa de producción petrolera (Q) aplicando la Ley de Darcy.
04 · Funciones
Qué son las funciones y cómo definirlas. Parámetros y retorno de valores.
Ejemplo práctico: Crear una función para calcular la presión hidrostática en un pozo de perforación.
05 · Manejo de Archivos
Leer y escribir archivos de texto (.txt) y archivos en excel.
Ejemplo práctico: Leer archivos de producción y calcular la producción promedio de un conjunto de pozos desde un archivo y generar un resumen.
06 · Listas y Diccionarios
Conceptos básicos: listas, diccionarios y sus aplicaciones.
Ejemplo práctico: almacenar datos de pozos (Nombre, Producción Diaria) y realizar cálculos simples.
07 · Resolución de Problemas Básicos
Aplicar conceptos aprendidos para resolver problemas específicos.
Ejemplo práctico: Calcular la tasa de producción de un pozo petrolero usando la ecuación de declinación exponencial.
08 · Proyecto Final + Simulación con las herramientas
Crear un script en Python para calcular el Índice de Productividad (IP), la tasa neta de producción de petróleo (Qo) y agua (Qw), además de determinar la Gravedad API de un crudo a partir de la gravedad específica (GEP).
Simulación con las herramientas de perforación, producción y yacimientos.
