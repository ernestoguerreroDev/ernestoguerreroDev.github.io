Herramientas digitales desarrolladas

Software a medida para optimizar decisiones técnicas y comerciales.

Simulador de Parámetros de Perforación

Gemelo Digital de un pozo a 8,500 pies. Genera perfiles geológicos, detecta eventos críticos (Kicks, Washouts) y visualiza Well Logs profesionales para reducir NPT.

Simulador de Producción

Integra Vogel con Machine Learning para IPR, declinación y límite económico. Predice producción futura con regresión lineal.

Simulador de Yacimientos

Modela fluidos en anticlinales, configura porosidad, permeabilidad y saturaciones. Recurso educativo interactivo para flujo fraccional.

Simulador de Circulación de un pozo

Simulando la Circulación de un pozo incluyendo Manómetro de presión y un tanque Activo, este proyecto nos recrea un sistema de circulación de un fluido integrando: Física de fluidos, Cálculos de Ingeniería Presión hidrostática, Volumen y Tiempo. Visualización interactiva: Pozo-Manómetro-Tanque.

Simulador de Circulación de un Pozo y Panel de Bomba

Herramienta interactiva para estudiantes de ingeniería petrolera donde podrán visualizar la circulación de un pozo con capacidades volumétricas en forma general para luego ir a casos reales y tener mayor aprendizaje en: Conceptos fundamentales de presión hidrostática, los volúmenes de tubería, anular y tiempo de circulación. Visualización de un panel de control de la bomba de lodo con parámetros críticos, cálculo de las RPM, presión y temperatura.

Simulador Avanzado de Parámetros de Perforación

Características: Aprendizaje mediante visualización de parámetros (ROP, Torque, RPM, Peso de Lodo, Presión Hidrostática, GR en tiempo real). Curva simulada de Gamma Ray en diferentes formaciones. Detección de problemas operacionales (Torque elevado, pérdida de presión) clasificados como Críticos o Advertencias. Gráficos interactivos con escalas logarítmicas y tooltips. Tecnologías: HTML5, CSS3, JavaScript, Chart.js, Anime.js, Bootstrap. Ideal para capacitar estudiantes y profesionales.

Simulador de Control de Pozos

Este simulador de Escritorio - Servidor permite a estudiantes de ingeniería petrolera conocer los conceptos generales con respecto al cálculo de presiones en el control de un pozo. Herramienta académica tecnológica con opciones a mejoras. Se puede escanear mediante código QR y visualizar en dispositivos móviles las variables en tiempo real. Desarrollada en: Python - Flask - JavaScript - Chart.js - Bootstrap.

Herramienta Para Análisis de Pruebas de Presión

Análisis de pruebas de presión utilizando la metodología de la curva maestra, modelado exponencial para ajustar automáticamente los parámetros críticos de la ecuación. Permite registrar datos como relación de movilidad (M) y parámetros (a,b) de la ecuación exponencial creando una curva única y representativa del yacimiento con diagnóstico gráfico incluyendo curvas de presión y derivada de presión.

Simulación de Producción Mediante un Equipo BES

Esta herramienta permite estudiar los conceptos de Drawdown y curvas de Declinación en un modelo visual, así como el movimiento de partículas de flujo y parámetros de producción (Caudal - Presión). Variables constantes: Pyac = 3500 psi, Qmax = 2900 BPD. Variables variables: Q, Pwf, IP (recalculado continuamente). Muestra curvas de declinación de Arps.
